# Configurações
* npm i -g typescript
* ou
* npm install -g typescript
## --------------------------
* tsc nomeArquivo.ts
* tsc --init inicar / configurações
* em config tsconfig.json 
* use assim apenas com essa configuração:

```javascript
{
  "compilerOptions": {
 
    "target": "ESNext",
    "strict": true
  }
}
```

* strict define que não podemos deixar tipo any implicitamente
## --------------------------
* tsc -w
